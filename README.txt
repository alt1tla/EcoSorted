1.Скачать git репозиторий.
2.Установить редактор кода (VS Code и т.п).
3.Создать в папке с проектом папку виртуального окружения.
4.Установить в неё Django и Pillow.
5.С помощью cmd открыть папку с проектом и запустить локальный сервер (python manage.py runserver).
6.Перейти по ссылке в cmd и откроется web-приложение.
